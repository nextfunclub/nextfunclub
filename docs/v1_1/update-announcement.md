# Next Fun Club v1.1

发布时间：2026.06.02

v1.1 重点提升活动发现效率和社交体验，让用户更快找到想去的活动，也能通过好友和聊天完成活动前沟通。

## 主要亮点

1. 活动发现页支持搜索、筛选和分页。
2. 活动详情页支持复制信息、生成宣传图和二维码。
3. 好友、收藏和私聊能力补齐活动前沟通场景。
4. 移动端好友与聊天页面更紧凑。
5. 添加好友前会先展示对方预览，减少加错人的情况。

## 用户可感知更新

1. 活动发现页支持关键词、主题、城市、活动形式和活动进度筛选。
2. 活动状态更清楚，可以区分可参与、已满员、进行中、未开始和已结束。
3. 活动列表支持分页，活动变多后也不会一次性加载过长页面。
4. 活动发现页加入每日推荐排序，同一天内展示顺序稳定。
5. 顶部支持全站搜索，可以搜索活动、地点和商家。
6. 活动详情页可以一键复制时间、地点、费用和活动链接。
7. 活动详情页可以生成带二维码的宣传图，方便下载后分享给朋友。
8. 活动可以收藏或取消收藏，常看的活动可以在个人空间快速找到。
9. 收藏按钮支持即时反馈，点击后不用等待整页刷新。
10. 用户可以添加好友，发送、接受、拒绝或取消好友申请。
11. 活动卡片和详情页会显示好友报名信号，帮助判断哪些活动更适合一起参加。
12. 好友页可以看到好友近期参加的公开活动。
13. 好友之间可以进入私聊，活动前可以简单确认时间、地点和安排。
14. 聊天列表会显示好友近期活动信号，方便从聊天入口继续约活动。
15. 聊天输入支持常用 emoji，聊天页也可以直接查看对方主页。
16. 移动端好友和聊天页面更紧凑，添加好友改为按钮和弹窗。
17. 添加好友前会先展示对方预览，确认目标后再发送申请。
