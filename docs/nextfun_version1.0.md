# Next Fun Club v1.0 更新公告

发布时间：2026.06.01

Next Fun Club v1.0 是第一版 MVP，重点完成“发现活动、报名参加、认识朋友、活动前沟通”的核心流程。

## 主要亮点

- 发现巴黎附近活动
- 创建和报名活动
- 添加好友并查看好友动态
- 私聊沟通活动安排

## 本次更新

1. 首页和活动页开始展示真实活动，打开页面就能看到最近可参加的活动。
2. 活动详情页展示时间、地点、人数、费用、发起人和报名状态，参加前可以先看清楚关键信息。
3. 登录后可以发起活动，填写标题、介绍、时间、地点、人数和费用后直接发布。
4. 可以报名参加活动，支持留言；需要审核的活动会先进入待确认状态。
5. 已经报名的活动可以取消，页面会同步更新人数和报名状态。
6. 个人空间可以查看个人资料、发起的活动、参与的活动和收藏的活动。
7. 活动发起人可以编辑活动信息，也可以取消活动。
8. 需要审核的报名可以由活动发起人通过或拒绝。
9. 系统会自动导入部分巴黎公开活动，活动库会持续补充新的公开活动信息。
10. 可以关注活动发起人，查看对方的关注和粉丝信息。
11. 活动详情页支持提问、建议和评价，方便参加前沟通细节。
12. 活动可以关联商家主页，查看场地、商家或组织者的公开信息。
13. 创建或编辑活动时可以上传封面图，让活动卡片和详情页更直观。
14. 活动地点支持地图展示，方便确认活动位置。
15. 报名、审核、活动取消和活动变更会进入通知中心，并在顶部显示未读提醒。
16. 创建活动时可以从活动链接导入信息，自动带出标题、时间、地点、封面和费用等内容。
17. 活动发现页支持关键词、主题、城市、活动形式和活动进度筛选。
18. 活动状态更清楚，可以区分可参与、已满员、进行中、未开始和已结束。
19. 活动列表支持分页，活动变多后也不会一次性加载过长页面。
20. 活动详情页可以一键复制时间、地点、费用和活动链接。
21. 活动详情页可以生成带二维码的宣传图，方便下载后分享给朋友。
22. 活动发现页加入每日推荐排序，同一天内展示顺序稳定，优先显示正在进行和即将开始的活动。
23. 顶部支持全站搜索，可以搜索活动、地点和商家。
24. 可以添加好友，发送、接受、拒绝或取消好友申请。
25. 活动卡片和详情页会显示好友报名信号，帮助判断哪些活动更适合一起参加。
26. 好友页可以看到好友近期参加的公开活动，快速发现朋友最近想去哪里。
27. 好友之间可以进入私聊，活动前可以简单确认时间、地点和安排。
28. 聊天列表会显示好友近期活动信号，方便从聊天入口继续约活动。
29. 移动端好友和聊天页面更紧凑，添加好友改为按钮和弹窗，不再占用主要浏览空间。
30. 聊天输入支持常用 emoji，聊天页也可以直接查看对方主页。
31. 添加好友前会先展示对方预览，确认目标后再发送申请，减少加错人的情况。
32. 活动可以收藏或取消收藏，常看的活动可以在个人空间快速找到。
33. 收藏按钮支持即时状态反馈，点击后不用等待整页跳转也能看到变化。
