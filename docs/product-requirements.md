# Product Requirements

## 产品定位

Next Fun Club 是一个面向海外中文用户的活动组织与找搭子平台。首发市场是法国巴黎，首发语言是中文。

产品第一阶段不做大型票务平台，也不做完整旅行攻略社区，而是验证一个明确闭环：

```text
发现活动 -> 查看详情 -> 发起活动 -> 报名参加 -> 管理活动
```

## 用户痛点

- 想约朋友出去玩，但微信群多、乱、信息难找。
- 想参加本地活动，但不知道哪里有人一起玩。
- 想发起饭局、看展、桌游、密室、短途旅行，但缺少结构化发布和报名工具。
- 想知道是否有熟人参与活动，从而提高组局成功率。

## MVP 功能范围

- 用户注册 / 登录
- 活动列表
- 活动详情
- 创建活动
- 编辑 / 取消自己发起的活动
- 报名参加活动
- 取消报名
- 活动人数统计
- 活动状态展示：招募中、已满、已结束、已取消
- 个人空间：我发起的活动、我参与的活动
- 移动端基础适配

## 非 MVP 功能

- 商家后台
- 支付
- 活动群聊
- 智能链接导入
- 复杂通知系统
- 推荐算法
- 多城市运营
- 完整多语言运营
- 地图选点
- 报名审核

## 后续验证问题

- 用户是否愿意用它，而不是继续用微信群？
- 用户更需要公共活动，还是朋友/熟人发起的活动？
- 活动卡片和报名机制是否足够支撑真实组局？
- 用户愿意为什么功能付费？
- 哪类活动最容易带来首批真实使用？
