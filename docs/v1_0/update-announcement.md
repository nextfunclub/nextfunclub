# Next Fun Club v1.0

发布时间：2026.06.01

v1.0 完成第一版 MVP 的核心闭环：用户可以发现活动、查看详情、发起活动、报名参加，并在个人空间管理自己的活动记录。

## 主要亮点

1. 真实活动数据开始进入首页和活动页。
2. 用户可以登录、发起活动、报名活动和取消报名。
3. 活动详情页展示时间、地点、费用、人数、发起人和报名状态。
4. 个人空间可以查看个人资料、发起的活动和参与的活动。
5. 活动发起人可以编辑活动、取消活动和处理报名审核。

## 用户可感知更新

1. 首页和活动页不再使用模拟数据，可以看到数据库中的真实活动。
2. 点击活动卡片可以进入完整活动详情页。
3. 登录用户可以创建活动，填写标题、介绍、时间、地点、人数和费用后发布。
4. 用户可以报名参加活动，也可以填写报名留言。
5. 需要审核的活动会先进入待确认状态，由发起人决定是否通过。
6. 已报名用户可以取消报名，页面会同步更新报名状态。
7. 活动发起人可以修改活动信息，或在计划变化时取消活动。
8. 个人空间展示个人资料、自己发起的活动和自己参与的活动。
9. 活动详情页支持基础提问、建议和评价，方便参加前沟通细节。
10. 活动可以关联场地、商家或组织者主页，帮助用户了解活动背景。
11. 活动可以上传封面图，活动卡片和详情页更直观。
12. 活动地点可以展示地图信息，方便确认位置。
13. 报名、审核、活动取消和活动变更会进入通知中心。
14. 创建活动时可以从活动链接导入标题、时间、地点、封面和费用等信息。
